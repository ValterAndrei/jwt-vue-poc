# JWT + Vue 3

POC com o objetivo de demonstrar autenticação usando Vue 3 com JWT ( Json Web Token )

Versão do node recomendada: ^20


### Setup com Docker

- Construir a imagem
```shell
docker compose build app
```

- Subir o container
```shell
docker compose up
```

- Acessar o container
```shell
docker compose run --rm app sh
```
